Pontos importantes a serem citados!

1 - Usei versão do Cypress 14.5.4
2 - Os elementos no portal estão fora do padrao sugerido para uso do Cypress, e por conta disso os testes foram criados de formas totalmente fora do padrão
podendo ocasionar erros na hora da execução dos mesmos. Ainda assim antes de subir pro git rodei todos eles e mandarei a evidencia no email.
3 - Devido a dificuldade no seletor dos elementos acredido que a primeira grande proposta para esse projeto seria de iniciar um trabalho de identificação dos elementos para somente 
depois termos melhores condições de automatizar os testes da melhor forma possível. Por exemplo colocando o padrão data-cy ficando: cy.get('[data-cy=nome-do-botao]').
4 - Qualquer dúvida estou a disposição para maiores esclarecimentos do código e da execução do mesmo.
